# Cyber Assessment Framework (CAF) Reference

This project is based on the **UK National Cyber Security Centre (NCSC) Cyber Assessment Framework (CAF)**.

The CAF is used by UK regulators and organisations operating Critical National Infrastructure (CNI) to assess cyber resilience and governance.

It supports the implementation of the **Network and Information Systems (NIS) Regulations** and is widely used across the UK public sector.

---

## Primary Objective Assessed in this Project

This simulated audit focuses on:

### Objective A – Managing Security Risk

The following principles were reviewed as part of the assessment:

A1 – Governance  
A2 – Risk Management  
A3 – Asset Management  
A4 – Supply Chain  

These principles examine whether organisations have appropriate governance, risk management processes, asset visibility, and supply chain security controls in place.

---

## CAF Structure Overview

The CAF is organised into four main objectives:

| Objective | Description |
|--------|-------------|
| A | Managing Security Risk |
| B | Protecting Against Cyber Attack |
| C | Detecting Cyber Security Events |
| D | Minimising the Impact of Cyber Security Incidents |

Each objective contains several **principles** and **Indicators of Good Practice (IGPs)** used to assess organisational maturity.
